# AIXLINX 物件検索 Chrome拡張機能 インストール手順

## インストール方法

1. Chromeで `chrome://extensions` を開く
2. 右上の **「デベロッパーモード」をON** にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. `chrome-extension/` フォルダを選択してOK

## 使い方

1. Chromeツールバーの拡張機能アイコンをクリック
2. お客さん一覧から選択（条件登録済みが上に表示）
3. **リアプロ / itandi BB / レインズ** を選択
4. 表示された検索条件を入力して物件を検索
5. 各条件の「コピー」ボタンでコピペ可能

## 条件の更新

お客さんの条件を変更したら、拡張機能の🔄ボタンで最新データを再取得できます。

## 注意

- 物件条件ページ（/conditions）で条件を登録したお客さんが「条件登録済み」として表示されます
- 条件未登録のお客さんは下部にグレー表示されます
