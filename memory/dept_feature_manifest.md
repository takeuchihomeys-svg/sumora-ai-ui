## 🗺️ 主要関数 依存マップ（popup.js）

> **#43-RG 管理**。この関数を変更するときは参照先を全部確認してからGOを出す。

| 関数/変数 | 参照している機能 | 変更時のリスク |
|----------|----------------|-------------|
| `renderInstrSteps(siteKey, cOverride)` | 3サイト全て・手順表示・一時調整更新 | **高** 全サイト影響 |
| `buildAdjCustomer(c)` | リアプロ/itandi/レインズ 自動入力条件生成 | **高** 3サイト影響 |
| `setupAreaModeSelector(c, siteKey)` | 3サイト共通・駅/地域切替ボタン | **高** 3サイト影響 |
| `preloadAdjForm(c)` | 3サイト共通・一時調整フォーム初期値 | 中 adj-form全体 |
| `STATION_LINE_MAP` | 駅/地域判定・リアプロ沿線・itandi路線・レインズ路線 | **最高** 全サイト全機能 |
| `ITANDI_LINE_MAP_FILL` | itandi BB路線モーダル自動操作のみ | 低 itandiのみ |
| `REINS_LINE_MAP` | レインズ路線名変換のみ | 低 レインズのみ |
| `LINE_STATION_ORDER` | 広げて検索の前後駅計算 | 中 広げて検索のみ |
| `NEIGHBORHOOD_WARD_MAP` | リアプロ詳細地域マッピング | 低 リアプロ地域のみ |
| `condオブジェクト` | popup.js → page-script.js 間通信 | **高** フィールド名変更厳禁 |

## ⚠️ 絶対に壊してはいけないもの（#43-RG 死守リスト）

```
1. STATION_LINE_MAP の既存エントリ（追加はOK・変更/削除は要確認）
2. condオブジェクトのフィールド名（desired_area / rent_max / walk_minutes / floor_plan / building_age）
   → page-script.js / itandi-page-script.js / reins-page-script.js が直接参照している
3. isUnderbar 判定ロジック（underbar.js と popup.js の両方に影響）
4. 3サイトのマッピング分離（リアプロ/itandi/レインズで路線名を共有しない）
5. clickItandiRadio() の label.click() 方式（inp.click()に戻すとMUI hiddenで壊れる）
6. ensureWardButtonY() の3段シーケンス（native.click→300ms→simulateClick→400ms→確認）
```
